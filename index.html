Kelir

About

Kelir is a 2D animation software written in C++, using Qt as its base application framework. Current development goal is a simple, easy to use, vector-based animation software.

Documentation

So far, there's only a short technical note that gives an overview of current state and planned development of Kelir. Its sole purpose is to help familiarize newcomer to Kelir's codebase.

Authors
